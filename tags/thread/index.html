Luồng & Đa luồng trong Java

⚙️ Đa luồng (Multithreading) trong Java

Đa luồng cho phép chạy nhiều tác vụ cùng lúc, giúp tận dụng CPU hiệu quả hơn.

Ví dụ:

class MyThread extends Thread {
  public void run() {
    System.out.println("Luồng đang chạy...");
  }
}

public class Main {
  public static void main(String[] args) {
    new MyThread().start();
  }
}
