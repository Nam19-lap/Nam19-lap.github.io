Ngôn ngữ lập trình Java

🗓️ 15/10/2025

Java là một ngôn ngữ lập trình hướng đối tượng mạnh mẽ, được phát triển bởi Sun Microsystems và hiện nay thuộc về Oracle.
Nó được sử dụng rộng rãi trong các ứng dụng doanh nghiệp, Android và hệ thống máy chủ.

Ưu điểm

  • Đa nền tảng (“Write Once, Run Anywhere”)
  • Có hệ sinh thái thư viện phong phú
  • Dễ dàng bảo trì và mở rộng

Ví dụ nhỏ

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello Java!");
  }
}
