Giới thiệu về cú pháp cơ bản trong Java

1. Tổng quan

Java là ngôn ngữ lập trình hướng đối tượng, kiểu tĩnh. Nó cũng độc lập với nền tảng — chương trình Java có thể được viết và biên dịch trên một loại máy, chẳng hạn như hệ thống Windows, và được thực thi trên một loại máy khác, chẳng hạn như MacOS, mà không cần bất kỳ sửa đổi nào đối với mã nguồn.

Trong hướng dẫn này, chúng ta sẽ xem xét và hiểu những điều cơ bản về cú pháp Java.

2. Kiểu dữ liệu

Có hai loại kiểu dữ liệu chính trong Java: kiểu dữ liệu nguyên thủy và kiểu dữ liệu đối tượng/tham chiếu .

Kiểu dữ liệu nguyên thủy là kiểu dữ liệu cơ bản lưu trữ dữ liệu đơn giản và tạo thành nền tảng cho việc thao tác dữ liệu. Ví dụ, Java có các kiểu dữ liệu nguyên thủy cho các giá trị số nguyên ( int, long, byte, short ) , giá trị dấu phẩy động ( float và double ) , giá trị ký tự ( char ) và giá trị logic ( boolean ).

Mặt khác, kiểu tham chiếu là các đối tượng chứa tham chiếu đến các giá trị và/hoặc các đối tượng khác hoặc đến giá trị đặc biệt null để biểu thị sự vắng mặt của giá trị.

Lớp String là một ví dụ điển hình về kiểu tham chiếu. Một thể hiện của lớp, được gọi là đối tượng, biểu diễn một chuỗi ký tự, chẳng hạn như “Hello World”.

3. Khai báo biến trong Java

Để khai báo một biến trong Java, chúng ta phải chỉ định tên (còn gọi là định danh) và kiểu dữ liệu của biến đó . Hãy xem một ví dụ đơn giản:

int a;
int b;
double c;

Java là ngôn ngữ lập trình hướng đối tượng (OOP) được phát triển bởi Sun Microsystems (nay thuộc Oracle).
Nổi tiếng với triết lý:

“Write once, run anywhere.”
— Viết một lần, chạy mọi nơi.

Java thường được sử dụng để phát triển:

  • Ứng dụng web (Spring Boot)
  • Ứng dụng Android (Java/Kotlin)
  • Hệ thống backend hoặc API lớn

💡 Ví dụ đầu tiên: Hello World

Đây là chương trình Java cơ bản nhất:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Xin chào, Java!");
    }
}

🧠 Giải thích:

public class HelloWorld : Khai báo một lớp tên là HelloWorld

public static void main : Hàm chính nơi chương trình bắt đầu

System.out.println() : Lệnh in ra màn hình

Khi chạy, Java sẽ hiển thị:

nginx Sao chép mã Xin chào, Java! ⚙️ Cấu trúc cơ bản trong Java Một chương trình Java thường có:

Lớp (Class) – mô tả đối tượng

Thuộc tính (Field) – dữ liệu của đối tượng

Phương thức (Method) – hành vi của đối tượng

Ví dụ:

public class Student {
    String name;
    int age;

    void study() {
        System.out.println(name + " đang học lập trình Java!");
    }
}

public class Main {
    public static void main(String[] args) {
        Student s1 = new Student();
        s1.name = "Nam";
        s1.age = 20;
        s1.study();
    }
}

Nam đang học lập trình Java!
