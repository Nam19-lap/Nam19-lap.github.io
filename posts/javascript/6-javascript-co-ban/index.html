JavaScript cơ bản — Ngôn ngữ của web

🗓️ 28/10/2025

🌐 JavaScript là gì?

JavaScript (JS) là ngôn ngữ lập trình chạy trên trình duyệt web, giúp trang web tương tác, động và thông minh hơn.

💬 HTML → cấu trúc
🎨 CSS → giao diện
⚙️ JavaScript → hành vi, logic

⚡ Ví dụ đầu tiên

<!DOCTYPE html>
<html>
<body>
  <h2>Xin chào JavaScript!</h2>
  <button onclick="sayHello()">Bấm vào đây</button>

  <script>
    function sayHello() {
      alert("Chào mừng bạn đến với JavaScript!");
    }
  </script>
</body>
</html>

🧠 Khi người dùng bấm nút → trình duyệt chạy đoạn code JS và hiển thị hộp thoại “Chào mừng bạn”.

🔍 Cách chèn JavaScript

Cách Cú pháp Mô tả
Nội tuyến <button onclick="alert('Hi!')">Click</button> Gắn trực tiếp vào HTML
Nội bộ <script> ... </script> Viết trong cùng file HTML
Ngoài file <script src="app.js"></script> Tách riêng file .js cho gọn gàng

🧩 Biến và kiểu dữ liệu

// Khai báo biến
let name = "Nam";
const age = 20;

// Kiểu dữ liệu phổ biến
let isStudent = true;          // boolean
let score = 9.5;               // number
let city = "Hà Nội";           // string
let subjects = ["JS", "HTML"]; // array
let info = { name: "Nam", age: 20 }; // object

console.log(name, age, city);

📘 Ghi nhớ:

  • let: có thể thay đổi giá trị

  • const: không thể gán lại

  • var: cú pháp cũ, nên hạn chế dùng

⚙️ Cấu trúc điều khiển

let score = 8;

if (score >= 9) {
  console.log("Xuất sắc");
} else if (score >= 7) {
  console.log("Khá");
} else {
  console.log("Cần cố gắng!");
}

for (let i = 1; i <= 3; i++) {
  console.log("Lần thứ " + i);
}

🔄 Hàm trong JavaScript

function greet(name) {
  return "Xin chào, " + name + "!";
}

console.log(greet("Nam"));

👉 Ngoài ra còn có hàm mũi tên (arrow function):

const greet = (name) => `Xin chào, ${name}!`;

🎯 DOM — Giao tiếp với HTML

JavaScript có thể truy cập & thay đổi nội dung HTML qua DOM (Document Object Model).

<h2 id="title">Xin chào!</h2>
<button onclick="changeText()">Đổi chữ</button>

<script>
  function changeText() {
    document.getElementById("title").innerHTML = "Chào từ JavaScript!";
  }
</script>
