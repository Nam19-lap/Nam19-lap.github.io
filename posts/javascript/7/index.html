Cấu trúc điều khiển trong JavaScript

🗓️ 30/10/2025

1. Câu lệnh điều kiện if / else

let age = 18;

if (age >= 18) {
  console.log("Đủ tuổi!");
} else {
  console.log("Chưa đủ tuổi!");
}

2. Câu lệnh switch

let day = 3;

switch(day) {
  case 1: console.log("Thứ Hai"); break;
  case 2: console.log("Thứ Ba"); break;
  default: console.log("Không rõ ngày!");
}

3. Vòng lặp for / while

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

let j = 0;
while (j < 3) {
  console.log("Lặp", j);
  j++;
}

🧩 Tóm tắt:

  • Cấu trúc điều khiển giúp kiểm soát luồng thực thi chương trình.

  • if – điều kiện; switch – nhiều trường hợp; for/while – lặp.
