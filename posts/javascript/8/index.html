Hàm trong JavaScript – Cách định nghĩa và sử dụng

🗓️ 30/10/2025

1. Hàm là gì?

Hàm (function) là khối mã thực hiện một nhiệm vụ cụ thể và có thể tái sử dụng.

function greet(name) {
  console.log("Xin chào, " + name);
}

greet("Nam");

2. Hàm mũi tên (Arrow Function)

const add = (a, b) => a + b;
console.log(add(2, 3)); // 5

3. Trả về giá trị

Hàm có thể trả về giá trị bằng từ khóa return:

function square(x) {
  return x * x;
}

console.log(square(4)); // 16

🧩 Tóm tắt:

  • Hàm giúp chia nhỏ chương trình, dễ tái sử dụng.

  • Sử dụng arrow function cho cú pháp gọn hơn trong ES6+.
